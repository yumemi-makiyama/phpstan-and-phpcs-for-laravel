# husky と lint-staged を理解するためのプロジェクト

## このリポジトリについて
Laravelに phpstan と php-csを導入して、コミット時に検査する仕組みを理解するために作りました。

仕組みは Qiitaの記事 「[husky と lint-staged 完全に理解した](https://qiita.com/akym03/items/7e4db720a9e1bfad7747) 」を
参照してください。
